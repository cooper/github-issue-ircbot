# github-issue-ircbot

```github-issue-ircbot``` responds to irc msg as like as ```#1234``` and replies github issue information.

## how to install

```
go install github.com/soh335/github-issue-ircbot
```

## usage

```
github-issue-ircbot --config /path/to/config.json
```
